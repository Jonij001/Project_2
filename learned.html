What I Learned

In this assignment I followed a lot of guides where I learned how the basics for d3 works and how to implement it (Link to main youtube tutorial used). One thing I had a hard time grasping was how to import a lot of different datasets and combine them using javascript. To then make a d3 visualization using many different datasets was hard for me. As I understood the d3-layout, I couldn't comprehend how to do a visualization from scratch. It was not until much later I saw we could use d3-templates, making the initial startup process easier.
